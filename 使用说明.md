# 使用说明（客户版）

## 1. 改文案

- 文件：`content.js`
- 说明：支持 `zh / ja / en` 三语言。请同步修改，避免切换语言后看到旧内容。

## 2. 改案例图片

- 图片目录：`assets/images/`
- 当前案例图：`li1.png / li2.png / li3.png`
- 图片路径由 `content.js` 的 `cases` 字段控制。

## 3. 改“精选网站与工具”模块（新增）

- 文件：`site-links.txt`
- 每行格式（推荐三语）：
  - `中文标题 | URL | 中文简介 | 日文标题 | 日文简介 | 英文标题 | 英文简介`
- 示例：
  - `养老金计算器 | https://example.com/pension-tool | 退休金测算页面示例 | 年金計算ツール | 年金試算ページの例 | Pension Calculator | Sample retirement calculation page`
- 规则：
  - 新增一行 = 前台新增一个卡片
  - 删除一行 = 前台删除一个卡片
  - 以 `#` 开头是注释，不会显示
  - 若只写前三列（中文标题/URL/中文简介），日文和英文会自动回退显示中文

## 4. 改样式

- 文件：`styles.css`

## 5. 改页面结构

- 文件：`index.html`

## 6. 本地查看

执行路径：`D:\Business\WebProjects\yara-agency-html-site`

```powershell
python -m http.server 8080
```

打开：`http://localhost:8080`

## 7. 提交给客户的文件

请只打包以下内容：
- `index.html`
- `styles.css`
- `content.js`
- `site-links.txt`
- `assets/`

不要包含本地缓存目录（如 `.wrangler/`）。
